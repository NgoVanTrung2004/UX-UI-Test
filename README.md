# UX-UI-Test
Bài test UX/UI – Thiết kế website bán hàng (Desktop)
## 1. Mô tả
Bài test UX/UI thiết kế giao diện website bán hàng phiên bản Desktop.
Giao diện sử dụng tiếng Việt hoàn toàn, tập trung vào bố cục rõ ràng,
luồng thao tác đơn giản và trải nghiệm người dùng.

## 2. Nội dung đã thực hiện
- UX Flow (sơ đồ luồng người dùng)
- Wireframe (giao diện trắng – đen)
- UI Visual Desktop
- Prototype tương tác
- Xuất file PDF tổng hợp
